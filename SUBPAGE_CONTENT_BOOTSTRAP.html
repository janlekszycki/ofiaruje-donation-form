WordPress Plugin

Wtyczka Ofiaruje dla WordPress

Osadź formularz darowizn ofiaruje.pl na swojej stronie i kieruj darczyńców bezpośrednio do bezpiecznej płatności.

Wtyczka jest publikowana publicznie na GitHub i GitHub jest głównym kanałem dystrybucji oraz aktualizacji.

Instalacja (GitHub)

  1. Wejdź do repozytorium: https://github.com/janlekszycki/ofiaruje-donation-form
  2. Pobierz ZIP z Code -> Download ZIP albo paczkę z Releases (zalecane).
  3. Upewnij się, że archiwum zawiera katalog ofiaruje-donation-form/.
  4. W WordPress przejdź do: Wtyczki -> Dodaj nową -> Wyślij wtyczkę.
  5. Wybierz ZIP i aktywuj plugin.

Konfiguracja

Po aktywacji przejdź do Ustawienia -> Ofiaruje i uzupełnij ID zbiórki oraz opcjonalne ustawienia UTM i wyglądu formularza.

Użycie shortcode

Wklej shortcode na stronie lub we wpisie:

[ofiaruje_formularz]
